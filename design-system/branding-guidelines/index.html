Branding guidelines

Make sure your service follows the required standards for branding so you don’t confuse users.

DfE blue department branding

Contents

  • Guidance
  • Code

When to use this style

  • if the service is provided on the education.gov.uk domain (internal or external users)
  • if it is an internal service such as an admin or caseworking service
  • if it falls outside the GOV.UK Proposition criteria

Do

  • use the departmental crest logo (Lion and Unicorn)
  • use Department for Education as the organisation name
  • use a cross-platform compatible font-stack
  • preferably use the departmental blue color scheme

Don’t

  • use the GOV.UK logotype
  • use the crown in the header
  • use the GDS Transport typeface (it’s not licensed for use outside GOV.UK)

Prototyping guidance

If using the GOV.UK Prototype Kit code section of the DFE blue department branding guidance.

Tell us if you’re using this branding in your service.

HTML

If you are using the GOV.UK Prototype Kit, replace the contents inside the {% block header %} {% endblock %} in the _layout.html file with the following HTML.

<header class="govuk-header app-header" role="banner" data-module="govuk-header">
  <div class="govuk-header__container govuk-width-container">
   <div class="govuk-header__logo">
     <a href="/" class="govuk-header__link govuk-header__link--homepage">
     <span class="govuk-header__logotype">
      <img src="/public/images/crest.png" alt="Department for Education homepage" class="govuk-header__logotype-crown-fallback-image">
        <span class="govuk-header__logotype-text">
          Department for Education |
        </span>
      </span>
      <span class="govuk-header__product-name">
        {{serviceName}}
      </span>
     </a>     
   </div>
   <div class="govuk-header__content">
    <button type="button" class="govuk-header__menu-button govuk-js-header-toggle" aria-controls="navigation" aria-label="Show or hide navigation menu">Menu</button>
    <nav>
      <ul id="navigation" class="govuk-header__navigation " aria-label="Navigation menu">
        <li class="govuk-header__navigation-item">
          <a class="govuk-header__link" href="#4">
            Sign out
          </a>
        </li>
      </ul>
    </nav>
  </div>
 </div>
</header>

Font stack

Add this to the top of your application scss or css file

This font stack has been designed to be compatible across browsers and platforms including mobiles and other devices.

body, p, h1, h2, h3, h4, h5, h6, a, span, div, table, th, td, blockquote, li, tr {
    font-family: 
        BlinkMacSystemFont,
        'Segoe UI',
        Roboto,
        Oxygen-Sans,
        Ubuntu,
        Cantarell,
        'Helvetica Neue',
        sans-serif !important;
}

Header style

Add this to your scss file. If using CSS, you’ll need to replace the variables with the color codes in the individual styles. 

$dfe-blue: #003a69;
$accent: #347ca9;

.dfe-template__body {

    .govuk-header {
        background-color: $dfe-blue;
    }
    
    .govuk-header__container {
        position: relative;
        margin-bottom: -10px;
        padding-top: 25px;
        padding-bottom: 15px;
        border-bottom: 10px solid $accent;
    }
    
    .govuk-header__product-name, 
    .govuk-header__logotype-text,
    .govuk-header__content {
        line-height: 2.5rem !important;
        font-size: 1.2rem;
    }
    
    .govuk-header__logotype-crown-fallback-image {
        height: 45px; 
        width: 45px;
    }
    
    .govuk-header__logotype {
        margin-right: 0;
    }
    
    .govuk-header__logo {
        @include mq ($from: desktop) {
          width: 70%;
        }
      }
      
    .govuk-header__content {
        @include mq ($from: desktop) {
            width: 30%;
            float: right;
        }
    }
    
    .govuk-header nav {
        display: inline-block;
        float: right;
    }
}

Crest logo (Lion and unicorn)

You must not use the GOV.UK Crown logotype on services not provided on GOV.UK or services.gov.uk. Instead, use the department crest.

If using the GOV.UK Prototype Kit, download the following crest file and save it to your /app/assets/images folder, and make sure it’s named crest.png

The crest image is white on a transparent background, so for display purposes is shown on the blue background.

Image will open in a new tab to save.

GOV.UK styles

Contents

  • Guidance
  • Code

When to use this style

  • if the service is public facing and transactional
  • provided on gov.uk or service.gov.uk

Do

  • use the GOV.UK page template
  • use the crown logo, GOV.UK logotype and GDS Transport typeface
  • use GOV.UK colors and styles

Prototyping guidance

This style used in the GOV.UK Prototype Kit. You can change it to the DfE blue by following the instructions in the code section of the DFE blue department branding guidance.
