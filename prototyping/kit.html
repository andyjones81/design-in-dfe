Prototype kit

Using the kit means you can build prototypes that look like a real GOV.UK service. This helps to make it easier to test for usability issues and get feedback on how the service might perform if it were a real service.

The kit is used across Government and supported thorugh the cross-gov #prototype-kit Slack channel.

There are tutorials and example code on the prototype kit website and you can make use of components and patterns from the GOV.UK Design System by copying HTML or Nunjucks code snippets into your prototype.

To be able to use the prototype kit, you’ll need to ensure your device has the following software installed:

  • Visual Studio Code (VS Code) - This is a code editor that enables you to make changes and run the kit as a website on your device.
  • Node.JS (LTS Version) - Node is a Javascript based framework for building websites and applications.
  • If you’re unfamiliar with using commands in a terminal to work with GitHub, you can download GitHub Desktop to make it easier.
  • If on a Windows device, you’ll need to install Git Bash

Branding your prototype

If your prototype is not intended for publishing to GOV.UK or service.gov.uk you should follow the branding your service guidance so that you do not cause confusion with your users.

Testing your prototype with users

Depending on what testing you plan to do with users, you might want to run your prototype as a website, rather than on your local device.

We use Heroku to host prototypes and this is free for basic usage and sufficient for most prototype testing. You’ll need to create an account using your DfE email address.

You’ll also need a GitHub account to store your code and enable it to be deployed to Heroku.

Prototypes should be created in the GitHub DFE-Digital organisation, clearly identifying them as a prototype. You can request access to GitHub through Digital Tools support.

Full instructions on how to set up a Heroku account and deploy your prototype can be found on the publishing to Heroku guidance on the GOV.UK Design System.
