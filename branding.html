Branding your service

You need to make sure the service uses the right header, footer and typeface for where it’s being provided, or you risk confusing users.

Services on gov.uk domains

You must follow the guidance on the GOV.UK Service Manual for making the service look like GOV.UK (opens in new tab) if your service is on one of the following domains:

  • gov.uk/myservice
  • myservice.service.gov.uk
  • myblog.blog.gov.uk

If the service is on education.gov.uk

You must make changes to your branding if you are using the GOV.UK prototype kit or GOV.UK Frontend and are providing your service on an education.gov.uk domain or your prototype is not intended to be a service provided on GOV.UK.

Examples of how to brand your service

Do
  • Use the departmental crest logo
  • Use Department for Education as the organisation name
  • Use a cross-platform compatible font-stack
Don’t
  • Don’t use the Crown or GOV.UK logotype
  • Don’t use the GDS Transport typeface
  • Don’t identify the service as being part of GOV.UK when it’s not

Example of a cross-platform compatible font-stack

body {
    font-family: -apple-system,
        BlinkMacSystemFont,
        'Segoe UI',
        Roboto,
        Oxygen-Sans,
        Ubuntu,
        Cantarell,
        'Helvetica Neue',
        sans-serif !important;
}

Example of the DfE header branding

HTML

If using the GOV.UK Prototype Kit, replace the content inside the {% block header %} block in the with the following HTML.

<header class="govuk-header app-header" role="banner" data-module="govuk-header">
   <div class="govuk-header__container govuk-width-container">
    <div class="govuk-header__logo">
      <a href="/" class="govuk-header__link govuk-header__link--homepage">
      <span class="govuk-header__logotype">
         <img src="/public/images/crest.png" class="govuk-header__logotype-crown-fallback-image">
          <span class="govuk-header__logotype-text">
            Department for Education
          </span>
        </span>
        <span class="govuk-header__product-name">
          Service name
        </span>
      </a>     
    </div>
  </div>
</header>

SCSS content to style the header


$dfe-blue: #003a69;
$accent: #4073ac;

.govuk-header {
    background-color: $dfe-blue;
}

.govuk-header__container {
    position: relative;
    margin-bottom: -10px;
    padding-top: 25px;
    padding-bottom: 15px;
    border-bottom: 10px solid $accent;
}

.govuk-header__product-name, .govuk-header__logotype-text {
    line-height: 2.5rem !important;
    font-size: 1.2rem;
}

.govuk-header__logotype-crown-fallback-image{height: 45px; width: 45px;}

Crest logo

You must not use the GOV.UK Crown logotype on services not provided on GOV.UK or services.gov.uk. Instead, use the department crest.

If using the GOV.UK Prototype Kit, download the following crest file and save it to your /app/assets/images folder, and make sure it’s named crest.png

The crest image is white on a transparent background, so for display purposes is shown on the blue background.
